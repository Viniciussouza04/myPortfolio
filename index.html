Verificando números

Um dos meus primeiros projetos, mas muito importante para mim, pois foi nela que comecei a ter a compreensão de como trabalhar com números através de uma linguagem de programação. A ideia do projeto é inserir um Input de números escolhido pelo usuário e a partir disso retornar informações sobre esses números, como qual deles é o maior, qual deles é o menor e até mesmo a soma de todos os números.
